# Marketing Product Performance Prediction Using Supervised Machine Learning

## Project Overview

This project predicts the Return on Investment (ROI) of marketing campaigns using supervised machine learning techniques. The objective is to identify the factors that influence marketing performance and compare different regression models to determine the best-performing model.

This project was developed as part of the Supervised Learning Assignment.

---

## Dataset

**Dataset Name:** Marketing and Product Performance Dataset

**Source:** https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset

**Dataset Size:**
- 10,000 Rows
- 17 Columns

**Target Variable:**
- ROI (Return on Investment)

---

## Project Objectives

- Predict marketing campaign ROI.
- Compare multiple supervised machine learning regression models.
- Identify the most important factors affecting ROI.
- Evaluate model performance using standard regression metrics.

---
